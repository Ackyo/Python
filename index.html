Python 常见出错原因及处理方式：

  1. 从cmd进入python idle打印中文报错，无论怎么设置coding都无济于事，但是自带的idle里面没有问题，原因：电脑上安装了ANSI166扰乱了编码 Solution：卸载ansicon，对于需要使用ansi166的程序，在打开命令行的时候让当前窗口有ansicon的功能即可
  2. 代码中有中文注释时报错：SyntaxError: Non-ASCII character '\xe7' in file D:\python\Test\hello\getjpg.py on line 13, but no encoding declared; see http://www.python.org/peps/pep-0263.html for details Solution： 第一行加上# -- coding: utf-8 --
